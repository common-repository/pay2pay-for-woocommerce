Плагин оплаты с помощью Pay2Pay для компонента магазина WooCommerce на базе Wordpress.

== Description ==

Данный плагин интеграции с Pay2Pay разработан для Wordpress WooCommerce 2.0.x.
Перед установкой модуля необходимо зарегистрироваться на сайте pay2pay.com (раздел "Подключиться") и в панели управления магазином заполнить следующие поля, значения которых используются при настройке плагина:
a) Секретный ключ - произвольный набор символов, используется для подписи сообщений при переадресации;
b) Скрытый ключ - произвольный набор символов, используется для подписи скрытых сообщений на Result URL;
c) Ключ API - произвольный набор символов, используется в случае дальнейшего расширения функций модуля разработчиками проекта.
Поля Success URL, Fail URL, Result URL заполнять не нужно.

Порядок установки модуля:
1. Выберите меню в консоли WordPress: Плагины / Добавить новый / Загрузить;
2. Выберите файл с архивом (wp-wc-pay2pay.zip) и нажмите "Загрузить";
3. В консоли WordPress выберите пункт меню  Плагины / Установленные, нажмите "Активировать";
4. Перейдите в меню WooCommerce -> Settings (Настройки) -> Payment Gateways (Способы оплаты), Вы увидите доступные способы оплаты. Нажмите на ссылку "Pay2Pay" для перехода к настройкам плагина;
5. Укажите все необходимые для работы с Pay2Pay настройки, аналогичные настройкамсвоего магазина в панели управления Pay2Pay (https://cp.pay2pay.com);
6. Установите тестовый режим работы.

Для проверки настройки системы необходимо провести тестовые платежи, денежные средства по которым сниматься не будут. Обратите внимание, что в панели управления магазином на сайте pay2pay.com статус успешно выполненных операций должен иметь значение "Выполнен".
Переключение в рабочий режим происходит после согласования с менеджером.
Тестирование работы плагина происходило вплоть до Wordpress 3.8 и WooCommerce 2.0.20.
